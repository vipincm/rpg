Prerequisites
- JDK 1.8
- Gradle 3.x

Build instructions: 
- ensure JAVA_HOME is set to JDK 1.8
- testing: gradle test
- build: gradle fatJar

Run instructions:
- ensure java 1.8 is in PATH
- run: java -jar build/libs/rpg-with-all-dependencies-0.0.1-SNAPSHOT.jar

View test results:
Open below file in web browser
- build/reports/tests/test/index.html
